Vite dev server isn't running

This is a debug build of Daisy. Debug builds load the React frontend from the local Vite dev server at http://localhost:5173, which isn't responding right now (ECONNREFUSED). Two ways to get going:

Option A · keep dev mode

Start Vite in another terminal

Then click Reload below.

cd ~/repos/smbr-daisy-app/apps/frontend
pnpm dev

Option B · ship-style

Use the release binary instead

Release builds embed a prebuilt frontend bundle and don't need a dev server. Quit this window, then: 

cd ~/repos/smbr-daisy-app
PATH="$HOME/.cargo/bin:$PATH" cargo build --release -p tauri-app
target/release/daisy-app &

You're seeing this page because the debug build's webview tried to load http://localhost:5173 and the connection was refused. The release build won't ever show it.
